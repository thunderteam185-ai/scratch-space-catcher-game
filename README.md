# Space Catcher Game (Scratch)

This project was built for the CS50 Scratch problem set.

## Game Concept
You control a spaceship that catches stars while avoiding meteors.

## Controls
Left Arrow – Move left  
Right Arrow – Move right  

## Features
- Score tracking
- Game over system
- Falling obstacles
- Custom Scratch blocks

## Built With
Scratch Programming Language
